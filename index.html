üìÑ POST /api/documents/analyze

Analyze document images via URL, base64, or file upload.

Option 1: URL (Recommended)

curl 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{"url": "https://example.com/ine.jpg"}'

Option 2: File Upload

curl 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze \
  -H "X-API-Key: your-api-key" \
  -F "file=@document.jpg"

Option 3: Base64

curl 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{"base64": "data:image/jpeg;base64,/9j/4AAQ..."}'
