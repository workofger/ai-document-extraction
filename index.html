Analyze with URL (Recommended)

Send just the image URL. The API will download and analyze automatically.

Request 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{"url": "https://example.com/document.jpg"}'

Analyze with File Upload

Upload JPG, PNG, WebP, or GIF files directly.

Request (multipart/form-data) 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze \
  -H "X-API-Key: your-api-key" \
  -F "file=@document.jpg"

Analyze with Base64

You can use /api/documents/analyze with base64 or the legacy endpoint /api/documents/analyze-base64.

Request (Unified endpoint) 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{"base64": "data:image/jpeg;base64,/9j/4AAQ..."}'
Request (Legacy endpoint) 
curl -X POST https://ai-document-extraction.vercel.app/api/documents/analyze-base64 \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "document": "data:image/jpeg;base64,/9j/4AAQ..."
  }'

Document Types

Type Description Extracted Data
INE Mexican Voter ID nombre, curp, claveElector, vigencia
LICENCIA Driver's License nombre, numero, tipo, vigencia
RFC Tax ID Certificate rfc, nombre, domicilio, cp
TARJETA_CIRCULACION Vehicle Registration placas, vin, modelo, propietario
POLIZA_SEGURO Insurance Policy aseguradora, poliza, vigencia
CLABE Bank Account Details banco, clabe, titular

Response Fields

Field Type Description
isValid boolean Document is valid and readable
detectedType string Detected document type
confidence number Confidence level (0.0 - 1.0)
extractedData object Data extracted from document
ocrEngine string hybrid | gpt-4o-vision
visionConfidence number Google Vision OCR confidence
imageQuality string buena | regular | mala | ilegible
illegibleFields array Fields that couldn't be read (marked with ***)
ocrCorrections array Applied corrections (e.g., O‚Üí0)
